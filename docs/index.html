🌍 Overview

CO2NEX is a decentralized platform for real-time carbon credit verification, certification, and offset — powered by AI, blockchain, and satellite intelligence.

We’re creating a new era of climate transparency, where every verified tonne of CO₂ is recorded, certified, and traceable from space to blockchain.

📁 What’s Inside (Planned)

  • frontend/ — React-based or SvelteKit dashboard UI
  • backend/ — API layer, database models, authentication
  • services/ — Integration with satellite data, IoT, GIS, etc.
  • docs/ — Technical documentation, architecture decisions
  • scripts/ — Deployment tools and DevOps utilities

🧱 Tech Stack (Work in Progress)

  • Frontend: React/SvelteKit + TypeScript
  • Backend: Node.js (NestJS) + Python (FastAPI)
  • Database: PostgreSQL + Redis
  • Blockchain: Ethereum + ZK Rollups (Polygon/StarkNet)
  • Auth: OAuth2 + Wallet Integration
  • Infra: Docker, CI/CD, Terraform
