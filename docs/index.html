🚀 This is the first public milestone of the CO2NEX Platform — the world’s first real-time carbon certification system powered by satellite data, AI, blockchain, and global validators.

🌍 Overview

CO2NEX is a decentralized infrastructure for carbon credit verification, certification, and offset, using real-time climate intelligence at planetary scale.

This repository hosts the core of our MVP and will include:

  • Core platform architecture
  • Frontend and backend services
  • Blockchain integrations
  • API / developer onboarding

📦 What’s Inside (Planned)

  • frontend/ — Dashboard UI
  • backend/ — API layer + database
  • services/ — Satellite data & AI pipelines
  • docs/ — Developer + API documentation
  • scripts/ — CI/CD and deployment tools

🧱 Tech Stack

  • Frontend: React or SvelteKit (TypeScript)
  • Backend: Node.js / Python (FastAPI)
  • Database: PostgreSQL + Redis
  • Blockchain: Ethereum + Polygon / StarkNet
  • Auth: OAuth2 + Web3 wallets
  • Infra: Docker, CI/CD, GitHub Actions
